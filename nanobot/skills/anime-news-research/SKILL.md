# 动漫资讯搜索调研

自动搜索并整理最新动漫资讯，生成结构化报告。

## 使用场景

当用户询问以下内容时触发：
- "最近有什么动漫新闻"
- "动漫资讯"
- "新番情报"
- "动画电影票房"
- "动漫行业动态"
- 任何与动漫、动画、ACG相关的新闻查询

## 工作流程

### 1. 搜索资讯

使用 `web_search` 工具搜索最新动漫资讯：

**搜索关键词组合：**
- "动漫新闻 最新" / "动漫新闻 2025"
- "新番情报" / "新番预告"
- "动画电影 票房"
- "国产动画" / "日本动画"
- 用户提到的具体作品名 + "新闻"

**搜索次数：** 执行 2-3 次不同关键词的搜索，确保信息全面

### 2. 获取详情

对重要新闻使用 `web_fetch` 获取详细内容

### 3. 整理报告

按以下结构整理资讯：

```
## 📰 动漫资讯快报（日期）

### 🔥 热点大事件
- 最重要的 1-2 条新闻

### 🎬 电影/剧场版
- 动画电影票房、上映信息

### 📺 TV动画/新番
- 新番情报、续作消息

### 🌟 国产动画
- 国产动画相关动态

### 📝 行业动态
- 公司新闻、政策、市场趋势

### 💡 其他趣闻
- 有趣的周边信息
```

### 4. 输出格式

- 使用 emoji 增加可读性
- 关键数据加粗
- 时间信息明确
- 保持简洁，每条新闻 1-3 句话

## 注意事项

1. **时效性**：优先展示最近 1-2 周的新闻
2. **准确性**：票房数据、上映日期等关键信息要准确
3. **多样性**：涵盖日本动画、国产动画、电影等多个维度
4. **语言**：根据用户语言习惯回复（中文用户用中文）
